# LEPG

> a Lightweight Edge Piercing Gateway

## 项目介绍

这是一个基于Golang的轻量级IoT边缘网关与内网穿透系统。包含一个边缘端和云端。

### 特色

在性能上，实现低配置需求。在使用上，自带内网穿透，配置简单且易于部署。

在边缘计算方面，该系统支持：

1. 本地缓存
2. 数据聚合
3. 断点续传

在功能上，支持不同协议，并支持自定义 TLV 格式进行粘包处理。

在支持完善的日志系统和实时监控功能的同时，支持性能监测。

### 优势

降低成本，提升效率，低内存需求，嵌入式友好。同时实现高并发处理的实例。自带内网穿透方便外网远程管理。

展示一定网络编程的深度以及工程化能力。

## 功能模块

### 通用

- 数据模型
- 日志：使用Go内置的Slog
- 配置：混合分层 + 影子同步

### 边缘端

- 内网穿透
	- 实现自研协议，模仿 FRP 自己做内网穿透，并集成 Yamux 库来实现多路复用。
	- 心跳保活
	- 连接管理、断线重连
- 读取输入数据，解析协议
- 整合数据
- 用~~MQTT~~自定义TCP隧道（就是上面自己实现的内网穿透）发到云端
	- sqlite本地缓存，断点续传

### 云端

- 监听边缘端连接+鉴权
- 连接管理（记录连接的设备）
- 心跳检测
- 用TLV进行协议解析
- 收数据，解析并存储
- 用长连接进行反向控制
- 控制台
